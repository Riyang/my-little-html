Test for Charcater Entity

Every HTML file should always has:

  • a <html> element
  • a <head> element
  • a <body> element

Here're tests for character entities:

This is the left-angle-bracket: <
This is the right-angle-bracket: >
This is the ampersand character: &

Here is the hello-world program in HTML:

<html>
  <head>
  </head>
  <body>
    <p>Hello, world!</p>
  </body>
</html>
